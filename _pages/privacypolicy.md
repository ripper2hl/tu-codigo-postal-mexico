# Política de Privacidad

**Última actualización:** 16 de Febrero de 2026

## 1. Descargo de Responsabilidad (Disclaimer)

Esta aplicación móvil (en adelante “la Aplicación”) proporciona información sobre códigos postales de México para facilitar la logística y ubicación de direcciones.

**AVISO IMPORTANTE:** La Aplicación “Códigos Postales para México” es una herramienta independiente y gratuita. **NO** está afiliada, asociada, autorizada, respaldada ni conectada oficialmente de ninguna manera con SEPOMEX (Servicio Postal Mexicano), Correos de México, ni con ninguna entidad gubernamental municipal, estatal o federal. Todas las marcas, nombres de productos y logotipos citados son propiedad de sus respectivos dueños.

## 2. Fuente de Información

Los datos mostrados en esta aplicación se obtienen de fuentes de acceso público y datos abiertos gubernamentales. Si bien nos esforzamos por mantener la base de datos actualizada (Versión 2026), no garantizamos la exactitud inmediata de los cambios realizados por las autoridades oficiales.

## 3. Uso Responsable

El usuario se compromete a utilizar la Aplicación con fines lícitos y de acuerdo con las leyes aplicables. Queda prohibido el uso de esta herramienta para actividades ilegales o fraudulentas.

## 4. Privacidad y Seguridad de los Datos

Respetamos su privacidad y cumplimos con las políticas de Google Play:

*   **No recopilación de datos personales por el desarrollador:** La Aplicación no requiere registro y el desarrollador no recopila ni almacena nombres, direcciones, números de teléfono ni correos electrónicos.
*   **Funcionamiento Offline:** La funcionalidad principal de búsqueda opera localmente en su dispositivo, minimizando la transferencia de datos.
*   **Publicidad de Terceros (Appodeal):** Para mantener la aplicación gratuita, utilizamos la plataforma de mediación de anuncios Appodeal. Appodeal y sus socios de red publicitaria pueden recopilar y utilizar datos técnicos y de uso, tales como el ID de publicidad de Android (AAID), dirección IP para ubicación aproximada e interacciones con los anuncios. Estos datos se utilizan exclusivamente para mostrar publicidad relevante y medir su rendimiento, de acuerdo con las políticas de privacidad de Appodeal.

## 5. Contacto

Si tiene alguna pregunta sobre esta Política de Privacidad, puede contactarnos a través de nuestra página de Facebook oficial o al correo electrónico de soporte proporcionado en la ficha de la tienda de aplicaciones.
